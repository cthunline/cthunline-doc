## Usage

### Development

```shell
# install dependencies
npm install
# run app for development
npm start
# run linters
npm run lint
# run linters with auto fix
npm run lint-fix
```

### Production

```shell
# install dependencies
npm install
# build app for production
npm run build
# serve built app
npm run serve
```
